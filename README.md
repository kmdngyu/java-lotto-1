# java-lotto
로또 미션 진행을 위한 저장소
---
###기능 요구 사항
- 구입 금액 입력
- 로또 수 입력
- 수동으로 구매할 번호 입력
    - 6개 숫자
    - ','로 구분
    -return으로 다음 수동 입력
- 번호 출력
- 지난 주 당첨 번호 입력
- 보너스 볼 입력
- 당첨 통계 출력
- 총 수익률 계산
